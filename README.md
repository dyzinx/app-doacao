📋 Sobre o Projeto
O App Doação é uma solução mobile nativa desenvolvida para facilitar o contato entre doadores e centros de coleta. O foco principal foi criar uma interface intuitiva e um fluxo de dados em tempo real.

🛠️ Tecnologias e Ferramentas
Linguagem: Kotlin

Ambiente: Android Studio

Backend: Firebase (Authentication e Realtime Database)

Design: Prototipado no Figma

🚀 Principais Desafios Superados
Implementação da lógica de autenticação segura.

Manipulação de listas dinâmicas com alto desempenho.
